# getConfig

## url
    http://{server}/getConfig/{configName}


## response (if success)

```json
{
    "RESULT_DATA": [
        {
            "config_value": 50
        }
    ],
    "RESULT_STATUS": "000",
    "RESULT_MESSAGE": "Successfull"
}

```
